**Meeting Description**: 

This side meeting explores emerging scenarios and technologies for distributed inference networks. We will discuss key technical gaps and potential standardization opportunities, with a focus on the evolving role of underlying networks—particularly edge gateways—in enabling inference acceleration, semantic routing, and adaptive resource orchestration. The session aims to foster cross-domain discussions and welcomes contributions from both networking and AI.
